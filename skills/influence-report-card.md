# Influence Report Card — Executable Skill

## Core Principle

Grade founder authority across 5 axes using objective rubrics and native platform data — not estimates, not feelings. Monthly 15-minute check-ins track progress against a 6-month blueprint. The entire report is an MAA cycle: Metrics (pull data), Analysis (explain why), Action (prioritized next steps).

## The 5 Axes

| Axis | Key Metrics |
|------|-------------|
| **Web Presence & SEO** | Monthly organic traffic, keywords top 10/50, Domain Authority, referring domains |
| **Public Speaking** | Engagements/year, venue quality, media mentions, testimonial videos |
| **Publishing** | Blog posts/year, guest posts, blog traffic, LinkedIn followers + engagement |
| **Podcasting** | Guest appearances/year, combined listener reach, owned podcast downloads |
| **Social Media** | Combined followers, avg engagement rate, posting frequency, growth rate |

## Grading Rubric (A-F per Axis)

| Grade | Web (organic visitors/mo) | Speaking (engagements/yr) | Publishing (blog traffic/mo) | Podcasting (appearances/yr) | Social (combined followers) |
|-------|--------------------------|--------------------------|-----------------------------|-----------------------------|---------------------------|
| **A** | >1,000 + DA >40 | 12+ incl. 3+ major | >10K + 5K LinkedIn | 12+ tier-1 shows | 10K+ + 5%+ engagement |
| **B** | 500-1,000 + DA 30-40 | 8-12 incl. 1-2 major | 2-5K + 2-5K LinkedIn | 8-12 quality shows | 5-10K + 3-5% engagement |
| **C** | 200-500 + DA 20-30 | 4-7 mostly webinars | 500-2K + 500-2K LinkedIn | 4-7 mid-tier | 2-5K + 1-3% engagement |
| **D** | <200 + DA <20 | 1-3 small platforms | <500 + <500 LinkedIn | 1-3 any | 500-2K + <1% engagement |
| **F** | No measurable traffic | 0 in last 12 months | No publishing activity | No podcast activity | <500 + no activity |

## Phase 0: Baseline Assessment (First Time — 2-4 hours)

1. Pull data from native platforms (GA4, SEMrush/Ahrefs, LinkedIn, podcast platforms, social analytics)
2. Grade each axis against rubric
3. Create 6-month blueprint with quarterly milestones (see template below)
4. Set end-state vision for each axis (realistic: most clients move 1-2 grades in 6 months)

## Phase 1: Monthly Check-In (15 minutes)

**Step 1 — Collect (5 min):** Pull fresh data from all platforms. Update tracking spreadsheet.
**Step 2 — Grade (5 min):** Update grade per axis. Note direction: improving, stable, declining. Use 3-month averages to smooth noise.
**Step 3 — MAA Summary (5 min):** Write the report:

```
# [Client Name] Influence Report Card — [Month Year]

## Current Grades
- Web Presence: [A-F] (was [grade] last month) [arrow]
- Public Speaking: [A-F] (was [grade] last month) [arrow]
- Publishing: [A-F] (was [grade] last month) [arrow]
- Podcasting: [A-F] (was [grade] last month) [arrow]
- Social Media: [A-F] (was [grade] last month) [arrow]
**Overall: [Grade] (trend: improving/stable/declining)**

## Key Metrics
| Axis | This Month | Last Month | Change | Target |
|------|-----------|-----------|--------|--------|

## Analysis: What's Working
[2-3 bullets with data]

## Analysis: Where It's Flat
[2-3 bullets with data]

## Recommended Actions (Next 30 Days)
1. [Tied to weakest axis]
2. [Maintain momentum in strongest axis]
3. [Quick win from Content Factory or promotion]

## Blueprint Progress
- Started at: [baseline] | Current: [now] | 6-month target: [goal]
- Trajectory: On track / Ahead / Behind
```

## 6-Month Blueprint Structure

**Q1 (Months 1-3) — Foundation:** Move overall grade up 1 level. Focus on biggest gaps.
**Q2 (Months 4-6) — Acceleration:** Compound gains. Scale what is working. Deprioritize axes that are not strategic.

Review blueprint quarterly. Adjust targets based on actual progress. Keep it visible.

## Common Scenarios

| Pattern | Diagnosis | Actions |
|---------|-----------|---------|
| Strong Web, Weak Social | Authority exists but not visible | Repurpose blog to LinkedIn, Dollar-a-Day on blog posts |
| Speaking Active, Publishing Dormant | Content Factory not running | Convert speaking audio to articles, set publishing as non-negotiable |
| Strong Social, Weak Web | Visibility but not owned | Drive social to website with CTAs, improve SEO, convert posts to blog |
| Stalled everywhere | Execution gap or unclear priorities | Pick ONE axis, create micro-goals, automate/outsource |

## Known Pitfalls

- **Grades based on feelings, not rubric.** Use the thresholds. Client pressure to inflate grades kills the tool's value.
- **Actions are vague.** "Increase blog traffic" is not an action. Make every action SMART: specific, measurable, assigned, realistic, time-bound.
- **Same weak axis month after month.** Decide: is it strategic or not? If yes, make it non-negotiable. If no, deprioritize and stop reporting it as a problem.
- **Report shows data but no business connection.** Add attribution: "This month's blog traffic came from X speaking engagement." Tie influence to outcomes.
- **Client sees grades but does not act.** Limit to TOP 3 actions. Offer specific help, not just recommendations.

## Human Gates

- Baseline data gathering (pulling from native platforms requires human access)
- Grading validated against rubric (not subjective impression)
- MAA analysis requires human judgment connecting cause to effect
- Monthly 15-minute client review call (maintains relationship, captures feedback)
- 6-month blueprint quarterly review (adjusting targets based on progress)
- Action item prioritization (limiting to top 3, ranking by impact/effort)
