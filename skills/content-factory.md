# Content Factory — Executable Skill

## Quick Decision: Which Stage?

- **New client, no tracking?** Start at Stage 0 (Plumbing)
- **Client ready to record?** Stage 1 (Produce)
- **Transcripts/raw content ready?** Stage 2 (Process)
- **Processed content needs publishing?** Stage 3 (Post)
- **Published content performing well organically?** Stage 4 (Promote)
- **Need weekly/monthly reporting?** Stage 5 (Perform)
- **Ongoing client?** Stages 2-5 run as a continuous loop

## Stage 0: Plumbing (One-Time Setup)

1. Verify GCT form completed
2. Set up: GBP, GA4, Search Console, remarketing pixels, UTM system
3. Create Content Library (Google Sheet) — central tracking tool
4. Set up content calendar and approval workflow (Draft > Client Review > Publish)
5. **HUMAN GATE:** Client provides business info and platform access; team lead verifies tracking

## Stage 1: Produce (Capture Raw Material)

1. Create Topic Wheel for client (content themes by expertise area)
2. Capture: Zoom calls, speaking events, podcasts, geotagged photos, one-minute videos
3. One-minute video formula: Hook (0-3s) > Ignite (3-15s) > Solution (15-50s) > CTA (50-60s)
4. Transcribe all video/audio within 48 hours
5. Log every asset in Content Library, tag by ACC stage and Topic Wheel category
6. **If client won't record:** Offer the 1-minute video formula (phone selfie, 60 seconds, one topic). If still stalled, mine existing content (past interviews, podcasts, webinars) before pressuring for new recordings.
7. **HUMAN GATE:** Owner MUST produce raw content — no one else can tell their stories

## Stage 2: Process (Transform into Assets)

1. Take transcript > use AI to create: articles, short-form clips, social posts, email segments, visuals
2. Articles: 1,200-2,000 words, E-E-A-T compliant, real stories woven in, keyword in first 100 words
3. Short-form video: 30-90 second segments, captions, branding, platform-optimized dimensions
4. Social posts: platform-specific (LinkedIn professional 100-150w, Facebook conversational, Instagram visual-first)
5. Run Article QA skill on all articles before handoff
6. **HUMAN GATE:** QA review of AI content, voice/tone approval, E-E-A-T compliance check

## Stage 3: Post (Distribute)

1. Plan 30 days ahead using Topic Wheel categories
2. Cadence: Blog 1-2x/week, Social daily on LinkedIn, Email 1-2x/week, GBP weekly
3. Channel-specific: WordPress (SEO settings, schema, mobile preview), YouTube (60-char title, custom thumbnail), LinkedIn (personal profile first, 7-9am), GBP (weekly, posts expire in 7 days)
4. Record all URLs in Content Library
5. **HUMAN GATE:** WordPress publishing, client approval for personal channels

## Stage 4: Promote (Dollar-a-Day)

1. Only promote content that hit organic thresholds (Video: 1,000+ views; Blog: 500+ visitors; Social: 100+ reactions)
2. Phase 1: $1-5/day for 7 days. Kill if CPC > $2 or engagement < 2%. Advance if CPC < $1 or engagement > 5%
3. Phase 2: $5-20/day for 30 days. Scale winners, expand targeting
4. Phase 3: Evergreen rotation $20-50/day. Refresh creative monthly
5. Run Thank You Machine: track mentions, ask to quote, send personalized gifts
6. **HUMAN GATE:** Budget approval at each phase, gift personalization

## Stage 5: Perform (MAA)

1. Weekly MAA every Friday: pull metrics from all platforms, analyze WHY, set actions with owners and deadlines
2. Score content on 30-point authority scale: What (1-10) x Who (1-10) x Where (1-10)
3. Winners (20+ score) go to evergreen promotion. Feed back into Process for repurposing
4. Kill or rework losers. Feed insights into Produce priorities
5. Monthly: full trend analysis, top 3 performers, Topic Wheel coverage gaps, client check-in
6. **HUMAN GATE:** Analysis requires human judgment, strategic pivots are human decisions

## Weekly Cycle Quick Reference

| Day | Focus | Stage |
|-----|-------|-------|
| Monday | Review analytics, plan captures | Perform > Produce |
| Tuesday | Capture/produce content, AI processing | Produce > Process |
| Wednesday | Client review and approval | Process |
| Thursday | Publish across channels, schedule social | Post |
| Friday | Promotion setup, Phase 1 testing, MAA report | Promote > Perform |

## Anti-Vandalism (Non-Negotiable)

- Never reduce the 6 stages to fewer
- Never remove MAA from any stage
- Never remove the Content Library requirement
- Never remove human-required designations
- Verify content tree before creating new articles (no cannibalization)
