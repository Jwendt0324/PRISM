## Overview

MAA (Metrics, Analysis, Action) is the performance measurement framework that runs at EVERY stage of the Content Factory. It is the "Perform" stage made systematic.

"In digital marketing, there is no penalty for taking many shots on the goal. We learn quickly and iterate constantly. We expect failure 90% of the time."

---

## The Three Components

### Metrics
- Pull from NATIVE platform analytics (Google Analytics, Facebook Insights, YouTube Studio, Search Console)
- Actual numbers, not estimates or approximations
- Week-over-week comparisons (absolute + percentage change)
- Key metrics per ACC funnel stage:
  - **Awareness:** Impressions, reach, video views, new followers
  - **Consideration:** Click-through rate, time on site, pages/session, video completion rate
  - **Conversion:** Leads, form submissions, calls, sales, CPA, ROAS

### Analysis
- Explain WHY the numbers are what they are
- Connect cause to effect ("Video views dropped 23% because we didn't post new content this week")
- Call out the WEAKEST area specifically
- This is NOT a task list — it requires human insight and judgment
- Dennis's test: "If your Analysis section reads like a list of what you did, it's wrong"

### Action
- Minimum 3 specific action items
- Each has: Owner + Deadline
- At least one action targets the weakest metric identified in Analysis
- Actions must be S.M.A.R.T. (Specific, Measurable, Achievable, Relevant, Time-oriented)
- Carried forward to next week's report until completed

| Report | Frequency | Due | Posted To |
|--------|-----------|-----|-----------|

## MAA Report Template

```
## Metrics (Week of [DATE])

### Awareness
- Website sessions: [X] (↑/↓ Y% WoW)
- Social reach: [X] (↑/↓ Y% WoW)
- New followers: [X]
- Video views: [X]

### Consideration
- Avg. time on site: [X] (↑/↓ Y% WoW)
- Blog post engagement: [X] comments, [X] shares
- Email open rate: [X]%

### Conversion
- Leads generated: [X]
- Calls received: [X]
- CPA: $[X]
- Revenue attributed: $[X]

## Analysis

[2-3 paragraphs explaining what's working, what's not, and why. Call out the weakest metric and hypothesize the cause.]

## Action

1. [Specific action] — Owner: [Name] — Deadline: [Date]
2. [Specific action] — Owner: [Name] — Deadline: [Date]
3. [Specific action targeting weakest metric] — Owner: [Name] — Deadline: [Date]

## Carried Forward (from last week)
- [Any incomplete actions from previous report]
```

## Common Violations

1. **Task lists disguised as Analysis** — "We posted 5 articles" is not analysis
2. **Missing numbers** — "Traffic increased" without saying from what to what
3. **No week-over-week comparison** — Single data points are meaningless
4. **No action targeting weakest area** — Actions all focus on strengths
5. **Skipping weeks silently** — If metrics are incomplete, post anyway with explanation
6. **Using third-party tool data** — Always use native platform analytics
